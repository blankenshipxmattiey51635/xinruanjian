# 个人炒股辅助软件（Windows 11 桌面版）

这是按阶段开发的 PC 炒股辅助软件。当前提交推进到**第二阶段**：在第一阶段主窗口、SQLite、配置和自选股管理基础上，接入免费公开真实行情源，支持 A 股/ETF/可转债/国内指数和美股/美股 ETF/美股指数的真实报价查询与定时刷新。

## 当前已实现

- Windows 桌面程序入口：`main.py`
- PySide6 主窗口，默认深色交易界面
- 首次启动设置向导：市场、刷新频率、扫描周期、AI 供应商、提醒方式、数据路径等
- SQLite 本地数据库：自选股、行情、新闻、AI 结果、策略、回测、模拟交易、设置、日志等表
- 自选股增删改查
- 真实行情源：
  - A 股、ETF、可转债、A 股主要指数：东方财富公开行情接口
  - 美股、美股 ETF、美股主要指数：Yahoo Finance 公开行情接口
- 自选股刷新：默认 5 秒，设置中可以调大，但最低保持 5 秒
- 大盘指数扫描：按设置的扫描周期定时扫描主要指数，并在日志页显示成功/失败
- 数据真实性保护：接口不可用时显示“数据源不可用/暂无真实数据”，不会生成随机行情或假行情
- API Key 本地加密保存工具，为后续 AI 模型接入预留
- 券商接口模块已预留但默认禁用，第一版不允许真实下单
- PyInstaller 打包脚本

## Windows 11 零基础运行步骤

1. 安装 Python 3.11 或更高版本。
2. 双击 `install.bat` 安装依赖。
3. 双击 `start.bat` 启动软件。
4. 首次启动会弹出风险提示和设置向导。
5. 在左侧输入股票代码，例如：
   - A 股：`600519`、`000001`、`300750`
   - ETF：`510300`、`159915`
   - 可转债：`110059`、`123022`
   - A 股指数：`000001.SH`、`399001.SZ`、`000300.SH`
   - 美股：`AAPL`、`MSFT`、`TSLA`
   - 美股指数：`IXIC`、`GSPC`、`DJI`、`RUT`、`VIX`
6. 选择市场后点击“添加/更新自选”。软件会按刷新周期请求真实行情。
7. 如果免费接口无法访问或不支持某个代码，表格会明确显示错误原因。

## 打包 exe

1. 先运行 `install.bat`。
2. 双击 `build.bat`。
3. 打包产物位于 `dist\StockAssistant\StockAssistant.exe`。

## 免费行情源限制

- 免费公开接口可能限流、变更字段、临时不可用或延迟。
- A 股盘中实时性取决于东方财富公开接口返回结果。
- 美股盘前盘后字段取决于 Yahoo Finance 当时是否返回 `preMarketPrice` / `postMarketPrice`。
- 如果接口不可用，本软件按要求显示错误，不会伪造数据。
