# 个人炒股辅助软件（Windows 11 桌面版）

这是按阶段开发的 PC 炒股辅助软件。当前提交完成**第一阶段**：项目结构、主窗口、SQLite 数据库、配置系统、首次启动向导、自选股管理，以及真实数据源不可用时的明确提示。

## 当前阶段已实现

- Windows 桌面程序入口：`main.py`
- PySide6 主窗口，默认深色交易界面
- 首次启动设置向导：市场、刷新频率、扫描周期、AI 供应商、提醒方式、数据路径等
- SQLite 本地数据库：自选股、行情、新闻、AI 结果、策略、回测、模拟交易、设置、日志等表
- 自选股增删改查
- 行情服务占位：尚未接入真实接口时显示“数据源不可用/暂无真实数据”，不会生成随机行情或假行情
- API Key 本地加密保存工具，为后续 AI 模型接入预留
- PyInstaller 打包脚本

## Windows 11 零基础运行步骤

1. 安装 Python 3.11 或更高版本。
2. 双击 `install.bat` 安装依赖。
3. 双击 `start.bat` 启动软件。
4. 首次启动会弹出风险提示和设置向导。
5. 在左侧输入股票代码，例如 `600519`、`AAPL`，选择市场后点击“添加/更新自选”。
6. 当前第一阶段不会显示真实价格；表格会明确显示数据源不可用。

## 打包 exe

1. 先运行 `install.bat`。
2. 双击 `build.bat`。
3. 打包产物位于 `dist\StockAssistant\StockAssistant.exe`。

## 风险提示

本软件第一版仅用于行情查看、研究分析、回测和模拟交易；AI 输出不构成投资建议；不接入真实券商交易，不允许自动真实下单。
